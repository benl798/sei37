Static (default)
The position is whatever the browser gives it, when it calculates the page flow; you can't change it later with top/left CSS offsets.
Relative Offsets do work, and are calculated relative to the default (static) position
Absolute Offsets work, and are calculated relative to the nearest containing element (ancestor) which has a specific position value set.... otherwise relative to the (whole document).
Absolute This has no positioned parent, so its offsets are calculated relative to the entire document/body.
Fixed Offsets are calculated relative to the window (viewport) - i.e. does NOT move when you scroll the document.
